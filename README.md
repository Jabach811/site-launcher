# Jabach811 Site Launchpad

A static GitHub Pages launcher for current published public sites.

## Included Sites

- Serenity Rose Studios: https://jabach811.github.io/marissa/
- CalCo Prints: https://jabach811.github.io/calco-prints-software/
- Selena Cherry: https://jabach811.github.io/selenas-site/
- Home Maintenance Checklist Binder: https://jabach811.github.io/household-checklist/
- Built From Within: https://jabach811.github.io/built-from-within-landing/

## Structure

- `index.html` is the complete standalone page.
- `assets/launchpad-hero.png` is the generated hero image used by the landing page.
- `.nojekyll` keeps GitHub Pages from applying Jekyll processing.

## Publish Notes

This repo is designed for legacy GitHub Pages from the `main` branch root.
